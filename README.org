
* 2sen-douka

** 二銭銅貨暗号ジェネレータ
   江戸川乱歩の『二銭銅貨』に出てくる南無阿弥陀仏暗号のジェネレータです。

** 使い方
   Github Pagesを用意したので [[https://boronology.github.io/2sen-douka/][こちら]] からどうぞ。

** サンプル
   「ゴケンチョーショージキドー」

   ↓

   「陀、無弥仏、南無弥仏、阿陀仏、弥、無阿弥陀、無陀、弥、無弥陀仏、無陀、陀、南無陀仏、南無仏、陀、無阿弥陀、無陀」
